# what is URL ?
  1. URL stands for universal or uniform resource locator 
  2. URL can open any website on web browser
  3. URL is open any website details n form of Home page and webpages 

# types of URL 

  There are 2 types of URL 
  1. absolute URL 
    
    **examples**

    ```
    https://www.tops-int.com/

    ```

  2. relative URL 
     
      **examples**

      ```
      https://www.tops-int.com/about
      or
      https://www.tops-int.com/contactus
      or
      https://www.tops-int.com/careers
      or
      https://www.tops-int.com/contact-us
      ```
