
# What is Browsers

A web browser is a software application used to access and view websites and web applications on the internet. It retrieves, renders, and displays content from web servers using web protocols (HTTP/HTTPS) and interprets HTML, CSS, and JavaScript.

## Types of Browsers

- Desktop graphical browsers: full-featured browsers for PCs and laptops.
	- Examples: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, Opera
- Mobile browsers: optimized for touch and small screens.
	- Examples: Chrome for Android, Safari on iOS, Samsung Internet, Firefox for Android
- Text-based browsers: render only text, used in terminals or limited environments.
	- Examples: Lynx, w3m, Links
- Headless browsers: run without a graphical interface; used for automated testing and scraping.
	- Examples: Puppeteer (headless Chrome), Playwright, PhantomJS (deprecated)
- Kiosk browsers: locked-down browsers for public terminals or single-purpose displays.
	- Examples: Kiosk mode in Chrome/Edge, SureFox
- Embedded browsers / WebView: browser components embedded inside apps.
	- Examples: Android WebView, WKWebView (iOS), CEF (Chromium Embedded Framework)

## Common browser features

- Tabbed browsing
- Bookmarking/favorites
- Private/incognito mode
- Extensions and add-ons
- Developer tools (console, inspector, debugger)

## Notes

- "Browser" is the correct spelling. This document lists common browser types and examples.
