# what is software

    Software is a set of instructions or programs that tell a computer what to do. It helps the computer perform different tasks.
    ```

    Example:

    ```

    Hardware = Mobile phone or computer (physical parts)
    Software = Apps and programs (WhatsApp, Chrome, MS Word)
    Types of Software

## 1.system software

Runs and controls the computer.
Helps the computer work properly.

    ```
    Examples:
    
    ```

    Windows
    Linux
    Android
    macOS
2. Application Software
Helps users do specific tasks.

Examples:

Microsoft Word (typing)
Google Chrome (browsing)
WhatsApp (chatting)
VLC Media Player (watching videos)
3. Programming Software
Used by programmers to create other software.

Examples:

Visual Studio Code
Eclipse
Python
GCC Compiler
4. Utility Software
Keeps the computer safe and fast.

Examples:

Antivirus
Disk Cleanup
Backup Software
WinRAR
5. Driver Software
Helps the computer communicate with hardware devices.

Examples:

Printer Driver
Graphics Driver
Audio Driver
6. Firmware
Software stored inside hardware that controls how it works.

Examples:

BIOS
Router Firmware
Smart TV Firmware
Easy Table
Type	Purpose	Example
System Software	Runs the computer	Windows, Linux
Application Software	Helps users do work	Word, Chrome, WhatsApp
Programming Software	Creates software	VS Code, Python
Utility Software	Protects and maintains the computer	Antivirus, Backup
Driver Software	Connects hardware to the computer	Printer Driver
Firmware	Controls hardware	BIOS, Router Firmware
Easy to Remember
System Software → Runs the computer.
Application Software → Helps users do tasks.
Programming Software → Makes other software.
Utility Software → Maintains and protects the computer.
Driver Software → Connects hardware to the computer.
Firmware → Built into hardware and controls it.
