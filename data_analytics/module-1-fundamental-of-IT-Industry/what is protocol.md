# Network Protocols

## 1. HTTP (HyperText Transfer Protocol)
- Purpose: Transfers web pages.
- Example: Browsing websites.

## 2. HTTPS (HyperText Transfer Protocol Secure)
- Purpose: Secure version of HTTP using encryption.
- Example: Online banking, shopping.

## 3. FTP (File Transfer Protocol)
- Purpose: Transfers files between computers.
- Example: Uploading website files.

## 4. SFTP (Secure File Transfer Protocol)
- Purpose: Secure file transfer.
- Example: Secure server file uploads.

## 5. TFTP (Trivial File Transfer Protocol)
- Purpose: Simple file transfer.
- Example: Network device configuration.

## 6. SMTP (Simple Mail Transfer Protocol)
- Purpose: Sends emails.
- Example: Gmail sending mail.

## 7. POP3 (Post Office Protocol Version 3)
- Purpose: Receives emails by downloading them.
- Example: Email clients.

## 8. IMAP (Internet Message Access Protocol)
- Purpose: Receives emails while keeping them on the server.
- Example: Gmail synchronization.

## 9. DNS (Domain Name System)
- Purpose: Converts domain names into IP addresses.
- Example: google.com → IP address.

## 10. DHCP (Dynamic Host Configuration Protocol)
- Purpose: Automatically assigns IP addresses.
- Example: Home Wi-Fi router.

## 11. TCP (Transmission Control Protocol)
- Purpose: Reliable data transfer.
- Example: Web browsing, email.

## 12. UDP (User Datagram Protocol)
- Purpose: Fast data transfer without guaranteed delivery.
- Example: Online games, video streaming.

## 13. IP (Internet Protocol)
- Purpose: Identifies devices and routes data.
- Example: IPv4, IPv6.

## 14. ICMP (Internet Control Message Protocol)
- Purpose: Reports network errors and diagnostics.
- Example: Ping command.

## 15. ARP (Address Resolution Protocol)
- Purpose: Finds MAC address from IP address.
- Example: Local network communication.

## 16. RARP (Reverse Address Resolution Protocol)
- Purpose: Finds IP address from MAC address.
- Example: Older network systems.

## 17. SSH (Secure Shell)
- Purpose: Secure remote login.
- Example: Server administration.

## 18. Telnet
- Purpose: Remote login (not secure).
- Example: Legacy systems.

## 19. SNMP (Simple Network Management Protocol)
- Purpose: Monitors network devices.
- Example: Routers, switches.

## 20. NTP (Network Time Protocol)
- Purpose: Synchronizes device time.
- Example: Internet time servers.

## 21. LDAP (Lightweight Directory Access Protocol)
- Purpose: Directory services and user authentication.
- Example: Active Directory.

## 22. SIP (Session Initiation Protocol)
- Purpose: Starts and manages voice/video calls.
- Example: VoIP.

## 23. RTP (Real-time Transport Protocol)
- Purpose: Transfers audio and video.
- Example: Video conferencing.

## 24. RTSP (Real-Time Streaming Protocol)
- Purpose: Controls media streaming.
- Example: IP cameras.

## 25. SMB (Server Message Block)
- Purpose: File and printer sharing.
- Example: Windows file sharing.

## 26. NFS (Network File System)
- Purpose: File sharing in Linux/Unix.
- Example: Shared network storage.

## 27. MQTT (Message Queuing Telemetry Transport)
- Purpose: Lightweight messaging for IoT.
- Example: Smart home devices.

## 28. AMQP (Advanced Message Queuing Protocol)
- Purpose: Message queue communication.
- Example: Enterprise applications.

## 29. CoAP (Constrained Application Protocol)
- Purpose: Communication for IoT devices.
- Example: Smart sensors.

## 30. BGP (Border Gateway Protocol)
- Purpose: Routes traffic between internet networks.
- Example: Internet backbone.

## 31. OSPF (Open Shortest Path First)
- Purpose: Internal network routing.
- Example: Enterprise networks.

## 32. RIP (Routing Information Protocol)
- Purpose: Dynamic routing.
- Example: Small networks.

## 33. EIGRP (Enhanced Interior Gateway Routing Protocol)
- Purpose: Cisco routing protocol.
- Example: Cisco enterprise networks.

## 34. PPP (Point-to-Point Protocol)
- Purpose: Direct communication between two devices.
- Example: Broadband connections.

## 35. L2TP (Layer 2 Tunneling Protocol)
- Purpose: VPN tunneling.
- Example: Secure VPN connections.

## 36. PPTP (Point-to-Point Tunneling Protocol)
- Purpose: Older VPN protocol.
- Example: Legacy VPNs.

## 37. IPSec (Internet Protocol Security)
- Purpose: Encrypts IP communication.
- Example: Secure VPN.

## 38. SSL (Secure Sockets Layer)
- Purpose: Secure internet communication (legacy).
- Example: Older HTTPS implementations.

## 39. TLS (Transport Layer Security)
- Purpose: Modern secure communication.
- Example: HTTPS websites.

## 40. WebSocket
- Purpose: Two-way real-time communication.
- Example: Online chat applications.

## 41. QUIC
- Purpose: Fast and secure internet transport.
- Example: HTTP/3.

## 42. XMPP (Extensible Messaging and Presence Protocol)
- Purpose: Instant messaging.
- Example: Chat applications.

## 43. SCP (Secure Copy Protocol)
- Purpose: Secure file copying.
- Example: Linux servers.

## 44. Rsync
- Purpose: Synchronizes files efficiently.
- Example: Backup systems.

## 45. mDNS (Multicast DNS)
- Purpose: Device discovery without a DNS server.
- Example: Apple AirPlay, printers.
